# Manuais e Treinamentos — Gestão Saúde Oral

Esta pasta concentra o planejamento, os roteiros de videoaula, as capturas de
tela e os manuais finais do Programa Sorriso da Gente.

## Estrutura

- `plano_mestre.md`: escopo, etapas, responsáveis e registro oficial do avanço;
- `ambiente_treinamento.md`: acesso, usuários fictícios, cenários e reset;
- `roteiros/`: textos para leitura durante a gravação de tela;
- `capturas/`: imagens aprovadas para os manuais;
- `manuais_pdf/`: fontes e PDFs finais;
- `videos/`: metadados, links e registros das gravações, sem armazenar
  obrigatoriamente os arquivos pesados no Git.

## Regra de atualização

Toda sessão de trabalho deve terminar com a atualização do
`plano_mestre.md`. O registro precisa indicar o que foi produzido, o que foi
validado, as pendências encontradas e o próximo passo.

Dados reais de pacientes não podem aparecer em vídeos, prints ou PDFs.
