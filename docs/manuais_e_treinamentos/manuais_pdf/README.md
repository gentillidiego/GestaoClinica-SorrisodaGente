# Manuais em PDF

Esta pasta receberá os arquivos-fonte em Markdown/HTML e os PDFs aprovados.

Cada manual deve informar:

- atividade e público;
- versão da aplicação;
- data da revisão;
- pré-requisitos;
- passos ilustrados;
- alertas e erros comuns;
- responsável pela validação.

## Referência complementar — SUS/SIGTAP

- [`docs/referencia_clinica_especialidades_procedimentos_sigtap.pdf`](../../referencia_clinica_especialidades_procedimentos_sigtap.pdf):
  referência institucional para Clínicos, com 8 especialidades e 133
  procedimentos;
- gerador: `scripts/generate_sigtap_clinical_reference.py`.
