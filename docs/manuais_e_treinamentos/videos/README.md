# Registros das Videoaulas

Registre aqui, em Markdown, os metadados de cada vídeo:

- título;
- atividade;
- versão da aplicação;
- data da gravação;
- apresentador;
- perfil utilizado;
- duração;
- link de armazenamento/publicação;
- status de revisão;
- observações e necessidade de regravação.

Arquivos de vídeo grandes não devem ser adicionados ao Git sem decisão
explícita sobre armazenamento e versionamento.
